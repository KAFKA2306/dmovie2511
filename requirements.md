## 目的
- 健康的な成人人物の映像を自動で再現性高く生成する。

## 対象
- ComfyUI と連携する自動化スクリプトおよびランタイム資産。

## フルオートワークフロー
1. ブリーフ受信でメタデータを読み込む。
2. テンプレートを合成し JSON ワークフローを生成する。
3. モデルと LoRA を検証し GPU ワーカーへ同期する。
4. レンダリングを実行し失敗時はシード変更で再試行する。
5. ポスト処理でエンコードと BGM 合成を行う。
6. カタログへ登録し通知を送る。

## システム構成
- オーケストレーション、アセット管理、監視を疎結合コンポーネントに分ける。
- ワークフロー定義からキューを生成し GPU 処理を追跡する。
- メタデータとログをバージョン管理し API で提供する。
- メトリクスと QA 結果をダッシュボード化する。

## 品質
- プロンプト、シード、モデル設定を成果物メタデータに保存する。
- `uv run pytest tests tests-unit` と `uv run ruff check . --fix` を必須実行とする。

## 導入
1. 承認フローを確立する。
2. パイロットワークフローを構築する。
3. 監視ダッシュボードを整備する。
